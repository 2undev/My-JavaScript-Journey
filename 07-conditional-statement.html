JavaScript conditional statements

What are JavaScript Conditional Statements?

Imagine you're playing a game with your friends. You might have rules like, "If it's raining, we play indoors.
If it's sunny, we play outside." In programming, conditional statements are like these rules.
They let your code make decisions based on certain conditions.



Why are Conditional Statements Needed?

Conditional statements are important because they allow your programs to make decisions.
Without them, your code would just do the same thing every time, no matter what.
But with conditional statements, your code can react differently based on different situations.



Why are They Important?

They are important because they make your code flexible and powerful. Think about apps you use every day. They all use conditional statements to behave differently based on what you do or what data they have. For example:
- A weather app shows different information based on your location and the current weather.
- A game might show different levels or actions based on your score or choices.

What are They Used For?

Conditional statements are used for:

  • Checking if certain conditions are true or false.
  • Making decisions and running different parts of your code based on these conditions.
  • Repeating actions until a certain condition is met.

How to Use Them

In JavaScript, the most common conditional statement is the `if` statement. Let's look at how it works with some relatable examples.

The `if` Statement

let weather = "sunny";
if (weather === "sunny") {
console.log("Let's play outside!");
}

Here, the code checks if the `weather` variable is equal to "sunny".
If it is, it prints "Let's play outside!" to the console.



The `if...else` Statement

Sometimes, you want to do one thing if a condition is true, and something else if it's not.

let weather = "rainy";
if (weather === "sunny") {
console.log("Let's play outside!");
} else {
console.log("Let's play indoors.");
}

Now, if the weather is "sunny", it says to play outside. Otherwise, it says to play indoors.



The `else if` Statement

You can also check multiple conditions using `else if`.

let weather = "cloudy";

if (weather === "sunny") {
console.log("Let's play outside!");
} else if (weather === "cloudy") {
console.log("Let's play board games inside.");
} else {
console.log("Let's watch a movie indoors.");
}

Here, if the weather is "sunny", you play outside. If it's "cloudy", you play board games inside. Otherwise, you watch a movie indoors.



The `switch` Statement

The switch statement is used for executing one block of code among many based on the value of an expression.
When you have many conditions to check, a `switch` statement can be cleaner.

let weather = "cloudy";

switch (weather) {
case "sunny":
console.log("Let's play outside!");
break;
case "cloudy":
console.log("Let's play board games inside.");
break;
case "rainy":
console.log("Let's watch a movie indoors.");
break;
default:
console.log("Let's stay in and read a book.");
}

Here, the `switch` statement checks the `weather` variable and runs the code for the matching case.

Other Things to Know

1. Comparison Operators:
These are used in conditions to compare values.

  • `===` checks if two values are equal.
  • `!==` checks if two values are not equal.
  • `>` and `<` check if one value is greater than or less than another.
  • `>=` and `<=` check if one value is greater than or equal to, or less than or equal to another.

2.Logical Operators: These are used to combine multiple conditions.

  • `&&` means "and". Both conditions must be true.
  • `||` means "or". At least one condition must be true.
  • `!` means "not". It inverts the condition.

Example with Logical Operators


let isWeekend = true;
let weather = "sunny";


if (isWeekend && weather === "sunny") {
console.log("Let's go to the beach!");
} else {
console.log("Let's stay home and relax.");
}

Here, you go to the beach only if it's the weekend and the weather is sunny.
